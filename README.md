# 名前
Discord-YouTube-to-mp3
YouTubeをダウンロードできるDiscordのbotです

# 使い方
!落とせ<YouTube等のURL>でダウンロードしてDiscordに送信してくれます

# 環境
Python3
Disord.py
youtube-dl
